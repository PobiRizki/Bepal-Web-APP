Memulai Menggunakan Firebase

Tutorial Memulai Menggunakan Firebase

pada tutorial di bawah ini merupakan, Tutorial untuk menambahkan data ke dalam firebase

1. Prepare data

Siapkan data anda yang ingin anda masukan ke dalam firebase disini saya menggunakan data pakain untuk kata kunci saya, sehinga, suatu saat saya akan menambahkan data lagi, saya bisa menambahkan kata kunci baru sesuai dengan apa yang kita inginkan

2. Store data

berikut merupakan salah satu contoh code java script untuk menambahkan data ke firebase bagian hitam pada gambar di atas merupakan prepare data, dan bagian merah merupakan bagian algoritma untuk menambahkan data ke firebase, tetapi ini hanya contoh code, bnyak contoh code yang laen yang bisa di gunakan untuk add data to firebase

3. firebase database rule

pada saat kita menambahkan data, tetapi data tidak masuk ke firebase. kenapa? karena firebase memiliki rule untuk database nya, rule nya yaitu harus login terlebih dahulu ke firebase, barulah bisa mengakses database tersebut. salah satu cara untuk tidak melakukan login adalah dengan mengubah rule tersebut

4. Skema no sql

perhatikan data yang telah berhasil anda masukan, pada konsep no sql. kita akan membuat root yang bernama "pakain" ini bertujuan untuk, jika ada yang bukan berkaitan dengan pakaian tidak di campurkan pada bagian data ini, selanjutnya, perhatikan data berwarna hitam, pada konsep no sql ini mirip seperti primary key, yang bertujuan untuk kemungkinan data "baju, celana, sepatu" bisa jadi sama, sehingga kita membuat semaca uID untuk membedakan, bahwasanya data nya ini emang berbeda. untuk kolom berwarna biru, merupakan data entry dari uID tertentu.

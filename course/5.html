Tutorial Memulai Menggunakan Firebase

pada tutorial di bawah ini merupakan, tutorial Menggunakan Firebase di halaman aplikasi web anda

1. Kunjungin halaman Firebase

anda dapat langsung ke halaman firebase disini . pilih menu GO TO CONSOLE yang ada pada jendela sebelah kanan atas website firebase untuk masuk ke dalam menu firebase

2. Add Project

pilih menu Add project, untuk menambahkan project yang baru anda kerjakan, masukan inputan form yang diminta

3. Singkron Firebase ke halaman kerja anda

Setelah itu, kita akan di suguhkan tampilan. ingin memulai menggembangkan aplikasi apa. ada 3 pilihan yang di sediakan dari firebase ios, android dan web app. klik tambahkan web app. lalu akan muncul pop up seperti di bawah ini.

4. Copy Code

copy code javascript yang di berikan firebase ke project anda, done. code anda telah tercoordinasi dengan firebase.
